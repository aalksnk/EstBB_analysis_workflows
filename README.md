# EstBB_analysis_workflows

This repository contains the computational workflows used to perform the analyses for the article "Leveraging nationwide health care records in Estonia to identify the genetic background of understudied disease phenotypes" ([preprint available]([url](https://www.medrxiv.org/content/10.1101/2025.03.18.25324091v2))).
The final version of the repository will be available upon publication in a peer-reviewed journal.
